# Alcaldia Calarca
El proyecto integra datos de accidentes, puntos críticos y parque automotor para crear una matriz de riesgo que identifique patrones y zonas prioritarias. Como grupo, desarrollamos una herramienta ágil que apoya decisiones y orienta acciones focalizadas de seguridad vial.
